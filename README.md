## Getting Started

### Prerequisites
* Flutter SDK
* Android SDK
* Android Studio/VS Code
* Git

### Installation

1. Clone the repo
```sh
git clone https://github.com/colt005/bored.git
```
2. Run the app on connected device
```
flutter run
```
